### _프로젝트 상세내용_
***
### 주제

```sh
TODAYS는 미니멀리즘을 추구하는 투두리스트입니다.
할 일을 효율적이고 직관적으로 정리할 수 있습니다.
```
<br>
<br>
    

### 배경

```sh
기존의 투두리스트 어플을 사용할 때 장기 일정 관리와 검색 기능이 필요할 때가 있었습니다. 
그래서 장기 일정을 날짜별, 요일별로 한번에 정리하고 관리할 수 있는 기능과
과거나 미래에 할 일에 대해 검색 기능도 있으면 좋을 것 같다는 생각을 했습니다.
```
